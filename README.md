# 菁优网组卷助手

这是一个可直接手动安装到 Chrome 的浏览器插件。

作用：

- 在 `https://www.jyeoo.com/*/ques/*` 和 `https://xyh.jyeoo.com/*/ques/*` 页面注入一个悬浮按钮
- 当页面存在可选题目时，显示 `全选本页题目`
- 点击后优先勾选页面里的题目复选框
- 如果页面不是标准复选框结构，则尝试点击文案接近 `选题 / 加入试卷` 的按钮
- 按钮会优先贴着页面右侧的 `试题篮`

## 安装

1. 打开 Chrome
2. 地址栏输入 `chrome://extensions/`
3. 打开右上角“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择当前文件夹：`/Users/shenchaonan/Documents/jyeoo_plugin`
6. 加载后会看到扩展名 `菁优网组卷助手`

## 使用

1. 打开菁优网的题目选择或组卷页面
2. 页面右侧 `试题篮` 附近会出现 `全选本页题目`
3. 点击一次后，会自动处理当前页可选题目

## 说明

由于菁优网页面结构可能按栏目或版本不同而变化，这个版本采用了“复选框优先、按钮文本兜底”的策略。

如果你后面能提供一张“具体组卷页面”的截图，或者给我那个页面的 URL，我可以继续把选择器改成更精准的定向版本，稳定性会更高。
