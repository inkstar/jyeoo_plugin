# 菁优网组卷助手

这是一个可直接手动安装到 Chrome 的浏览器插件。

## 功能

- 在 `https://www.jyeoo.com/*/ques/*` 和 `https://xyh.jyeoo.com/*/ques/*` 页面注入组卷助手
- 支持 `全选本页试题`
- 支持 `选择多页试题`，跨页自动续跑，默认目标为 `60` 题
- 会优先点击题目旁边的 `试题篮` 按钮，并跳过已加入状态
- 悬浮模块支持拖拽、隐藏/展开，并会记住位置与折叠状态
- 带有常驻滚动日志面板，便于观察当前处理进度和翻页状态

## 项目结构

- `manifest.json`：Chrome 扩展入口配置
- `content.js`：页面注入、选题逻辑、跨页逻辑、拖拽/隐藏逻辑
- `content.css`：悬浮模块样式
- `scripts/test-plugin.js`：本地浏览器自动化调试脚本

## 安装

1. 打开 Chrome
2. 地址栏输入 `chrome://extensions/`
3. 打开右上角“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择当前文件夹：`/Users/shenchaonan/Documents/jyeoo_plugin`
6. 加载后会看到扩展名 `菁优网组卷助手`

## 使用

1. 打开菁优网题目页
2. 页面右侧 `试题篮` 附近会出现组卷助手
3. 点 `全选本页试题` 会处理当前页题目
4. 点 `选择多页试题` 会自动翻页继续选题，直到题篮达到目标数量或没有下一页
5. 可拖动顶部工具条调整位置，点右上角按钮可以隐藏/展开

## 调试

- 页面内日志会显示在悬浮面板中
- 开发者工具 Console 中可搜索前缀 `JYEOO-PLUGIN`
- 如需本地自动化调试，可运行 `node scripts/test-plugin.js`

## 说明

菁优网页面结构会按栏目、登录态、筛选方式变化，所以当前实现采用了“显式 `试题篮` 按钮优先，其他交互兜底”的策略。

如果你后面能提供一张“具体组卷页面”的截图，或者给我那个页面的 URL，我可以继续把选择器改成更精准的定向版本，稳定性会更高。
